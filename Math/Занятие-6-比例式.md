### Expression

- 比例式的性质
  - 中间项的乘积等于两边项的乘积
  - 交换中间项仍成立
  - 交换两边项仍成立
  - 分别交换等式两边的分子分母仍成立
  - 交换等式两边的比例式仍成立
- Если а : b = с : d
  - a : с = b : d
  - d : b = с : а
  - b : a = d : c
  - с : d = a : b
  - b : d = a : c
  - с : а = d : b
  - d : с = b : a
- сколько процентов составляет а от т
