### Expression

- 无理不等式解法：
  - ①√(f(x)) < φ(x)
  - 满足以下情况时有解：
    - f(x) ≥ 0
    - φ(x) > 0
    - f(x) < φ<sup>2</sup>(x)
  - ②√(f(x)) > φ(x)
  - 当φ(x) < 0 时，且满足以下情况时有解：
    - f(x) ≥ 0
    - φ(x) < 0
  - 当φ(x) ≥ 0 时，且满足以下情况时有解：
    - φ(x) ≥ 0
    - f(x) ≥ 0
    - f(x) > φ<sup>2</sup>(x)
  - ③√(f(x)) ≤ φ(x) 
  - 满足以下情况时有解：
    - f(x) ≥ 0
    - φ(x) ≥ 0
    - f(x) ≤ φ<sup>2</sup>(x)
  - ④√(f(x)) ≥ φ(x)
  - 当φ(x) < 0 时，且满足以下情况时有解：
    - f(x) ≥ 0
    - φ(x) ≤ 0
  - 当φ(x) ≥ 0 时，且满足以下情况时有解：
    - φ(x) ≥ 0
    - f(x) ≥ 0
    - f(x) ≥ φ<sup>2</sup>(x)
