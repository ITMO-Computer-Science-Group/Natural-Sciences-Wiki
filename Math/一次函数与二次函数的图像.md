### Expression

- 一次函数
  - 形式：y = ax + b : R → R
  - 模式：y = ax + b : 定义域 → 值域
- 二次函数
  - 一般式：y = ax<sup>2</sup> + bx + c : R → R
  - 顶点式 y=a(x−h)<sup>2</sup>+k
    - h=b/2a
    - k=(4ac−b<sup>2</sup>)/4a
    - 最值：k
    - 对称轴：h
  - 韦达定理：x<sub>1</sub>+x<sub>2</sub>=−b/a  x<sub>1</sub>·x<sub>2</sub>=c/a
  - 求根公式：x<sub>1,2</sub>=(-b±√Δ)/2a
